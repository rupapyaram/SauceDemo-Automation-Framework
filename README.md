# SauceDemo Login Automation Framework

## Project Overview

This project automates the Login Module of SauceDemo using:

* Java
* Selenium WebDriver
* TestNG
* Maven
* Page Object Model (POM)

## Automated Test Cases

1. Valid Login Test
2. Invalid Login Test
3. Locked User Login Test

## Framework Features

* Page Object Model (POM)
* TestNG Assertions
* BeforeMethod and AfterMethod
* Maven Dependency Management

## Tools Used

* Java
* Selenium WebDriver
* TestNG
* Maven
* Eclipse
* GitHub
